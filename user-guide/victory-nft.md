# Victory NFT

Victory NFTs are unique collectibles minted to solvers who successfully solve puzzles on TreasureHunt.

## What is a Victory NFT?

A Victory NFT is an ERC-721 token that serves as:

- **Proof of achievement** - You solved a puzzle!
- **Permanent record** - Stored on-chain forever
- **Collectible item** - Can be viewed in your profile

## Victory NFT Metadata

Each Victory NFT contains the following information:

| Field | Description |
|-------|-------------|
| Riddle ID | The puzzle you solved |
| Creator | Puzzle creator's address |
| Riddle Content | The puzzle prompt |
| Answer | The correct answer |
| Solver | Your address (the winner) |
| Solve Time | When the puzzle was solved |
| Duration | Time taken (in blocks) |
| Reward | Prize amount won |

## Viewing Your Victory NFTs

### In the dApp

1. Navigate to your Profile page
2. Look for the "Victory NFT Gallery" section
3. All your Victory NFTs will be displayed as cards
4. Click on any card to view full metadata

### On BscScan

You can also view your Victory NFTs on BscScan:

1. Go to [bscscan.com](https://bscscan.com)
2. Navigate to the "NFT" section
3. Enter your wallet address
4. View your NFT holdings

## Token ID System

TreasureHunt uses a specific token ID scheme:

- **Puzzle NFTs**: Token IDs 0, 1, 2, 3...
- **Victory NFTs**: Token IDs 1,000,000,000, 1,000,000,001, ...

This ensures no conflicts between puzzle and victory NFTs.

## Transferring Victory NFTs

Victory NFTs are standard ERC-721 tokens, meaning:

- You can transfer them to any address
- They can be bought and sold on NFT marketplaces
- They will always show your achievement as the original solver
