# Getting Started

This guide will help you get started with TreasureHunt in just a few minutes.

## Prerequisites

Before using TreasureHunt, you need:

1. **A Web3 Wallet** - MetaMask is recommended
2. **BNB Balance** - For creating puzzles or participating
3. **Correct Network** - BSC Mainnet or opBNB Mainnet

## Step 1: Install a Web3 Wallet

If you don't have a Web3 wallet yet:

1. Download **MetaMask** from [metamask.io](https://metamask.io)
2. Install the browser extension or mobile app
3. Create a new wallet or import an existing one
4. Secure your seed phrase!

## Step 2: Add BNB Chain Network

TreasureHunt runs on BNB Chain. Add the network to MetaMask:

### BSC Mainnet
- **Network Name**: BSC Mainnet
- **RPC URL**: https://bsc-dataseed.binance.org
- **Chain ID**: 56
- **Symbol**: BNB
- **Block Explorer**: https://bscscan.com

### opBNB Mainnet
- **Network Name**: opBNB Mainnet
- **RPC URL**: https://opbnb-mainnet-rpc.bnbchain.org
- **Chain ID**: 204
- **Symbol**: BNB
- **Block Explorer**: https://opbnbscan.com

## Step 3: Get Some BNB

You'll need BNB to:

- Create puzzles (minimum 0.01 BNB)
- Participate in puzzles (deposit × multiplier)
- Pay for gas fees

You can get BNB from:

- Central exchanges (Binance, Coinbase)
- Decentralized exchanges
- P2P platforms

## Step 4: Access TreasureHunt

Open your browser and navigate to the TreasureHunt dApp:

```
https://treasurehunt.example.com
```

> Note: Replace with the actual deployed URL

## Step 5: Connect Your Wallet

1. Click "Connect Wallet" in the navigation
2. Approve the connection request in MetaMask
3. Your address will be displayed in the navbar
