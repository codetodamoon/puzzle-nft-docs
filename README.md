# TreasureHunt - Blockchain Puzzle NFT dApp

Welcome to the TreasureHunt documentation. This comprehensive guide covers everything you need to know about using and developing on the TreasureHunt platform.

## What is TreasureHunt?

TreasureHunt is a decentralized application (dApp) built on EVM-compatible chains (BSC, opBNB, Ethereum) where players can create cryptography puzzles with BNB prizes. Other players can participate by solving these puzzles to win the prize pool and earn a unique Victory NFT.

## Key Features

- **Create Puzzles** - Set your own riddles with BNB deposits and custom multipliers
- **Solve Puzzles** - Lock puzzles and submit answers to win prizes
- **Victory NFT** - Earn unique NFTs as proof of your puzzle-solving achievements
- **PNL Tracking** - Track your earnings and performance
- **Decentralized** - All transactions are secured by smart contracts

## Supported Networks

- BSC Mainnet
- opBNB Mainnet
- BSC Testnet (for development)
- opBNB Testnet (for development)
