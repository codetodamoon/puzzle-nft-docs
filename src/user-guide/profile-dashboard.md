# Profile & Dashboard

Your profile page is your personal dashboard on TreasureHunt, showing all your activity and performance.

## Accessing Your Profile

Click on your connected wallet address in the navigation bar, or navigate to `/profile`.

## Overview Statistics

Your profile displays key metrics:

### As a Puzzle Creator

| Metric | Description |
|--------|-------------|
| Created | Total puzzles you've created |
| Solved | How many were solved by others |
| Timeout Wins | Times solvers failed to answer |
| Total Earned | Sum of all prizes won |

### As a Solver

| Metric | Description |
|--------|-------------|
| Attempts | Total puzzles you've attempted |
| Solved | Number of puzzles solved |
| Win Rate | Percentage of successful solves |
| Total Won | Sum of all prizes won |

### Overall PnL

| Metric | Description |
|--------|-------------|
| Net PnL | Total earnings minus total deposits |
| Total In | All deposits made |
| Total Out | All prizes won |

## My Puzzles

The "My Puzzles" section shows puzzles you've created:

- Current status (Joinable, Decrypting, Solved, Timeout)
- Puzzle details (deposit, multiplier)
- Current solver (if locked)
- Actions available (edit, destroy, withdraw)

## Victory NFT Gallery

All Victory NFTs you've earned are displayed here:

- Each NFT card shows key info
- Click to view full metadata
- Links to view on BscScan

## PnL History

A chronological list of your transaction history:

- Puzzle creation events
- Solving attempts
- Wins and losses
- Timestamps and amounts

## Filtering and Searching

Use filters to find specific records:

- By type (creation, solving, win, loss)
- By status (active, solved, timeout)
- By date range

## Connecting Your Wallet

If your wallet isn't connected:

1. Click "Connect Wallet"
2. Approve the connection in MetaMask
3. Your profile data will load automatically
